# 홈페이지 톤에 맞춘 프로젝트 카드 스타일

프로젝트 카드의 본문 배경을 기존 흰색에서 홈페이지의 네이비/블루 계열 다크 톤으로 변경했습니다.

수정 파일:

```text
assets/css/style.css
```

주요 선택자:

```css
.project-card
.project-card-body
.project-card h3
.project-period
.project-category
.project-summary
.tag-list li
.project-card-footer
.card-button
```

카드 배경을 더 밝게 또는 어둡게 조절하려면 `.project-card`의 `background` 값을 수정합니다.
